# Home
url : trello.com/**home**

로그인을 클릭하면 [**Login**](#login) 페이지로 이동.

# <a name="login"></a>Login
url : trello.com/**login**

로그인 페이지. 이메일 + 비밀번호 로그인과 구글 로그인 가능.

# Boards
url : trello.com/**[name]**/**boards**

* 즐겨찾기 설정한 보드 리스트 (Starred Boards)
* 개인 보드 리스트 (Personal Boards)
  * 보드 생성
* 팀 보드 리스트 ([Team name])
  * 보드 생성
